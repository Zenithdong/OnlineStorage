## 🔌 通信协议

所有数据包采用 **长度前缀格式**，结构如下：

```
┌─────────────────┬────────────────────────────────────────┐
│   Length (4B)   │              Payload (N bytes)         │
│   int32 大端     │   STRU_BASE 子类，通过 m_nType 路由     │
└─────────────────┴────────────────────────────────────────┘
```

| 协议号 | 方向 | 功能 | 状态 |

## ⚡ 快速开始

### 📋 环境要求

```
✔ Windows 10/11
✔ MinGW-w64 (g++ 支持 C++17)
✔ MySQL Server 8.0
✔ VSCode (可选，推荐)
```

### 🔨 构建项目

**方式一：VSCode 一键构建**
```
按下 Ctrl + Shift + B
```

**方式二：CMake**
```bash
mkdir build && cd build
cmake ..
cmake --build .
```

**方式三：手动编译**
```bash
g++ -g \
  Server/main.cpp \
  Server/netWork/tcpnet.cpp \
  Server/CMySQL/cmysql.cpp \
  Server/Kernel/kernel.cpp \
  -o Server/Server.exe \
  -lws2_32 \
  -I"C:/Program Files/MySQL/MySQL Server 8.0/include" \
  -L"C:/Program Files/MySQL/MySQL Server 8.0/lib" \
  -lmysql
```

### ▶️ 启动服务

```bash
# 1. 确保 MySQL 服务运行中
# 2. 修改数据库配置（默认如下）
#    → Server/Kernel/kernel.cpp:39
#    → ConnectMySql("127.0.0.1", "root", "密码", "server")

# 3. 启动服务端
./Server/Server.exe

# 服务端将监听 → 0.0.0.0:8899
```

# 网盘项目 (OnlineStorage)

## 项目简介

这是一个基于 Windows 的**网盘服务器**项目，提供用户注册、登录和文件管理功能。

### 技术栈

| 组件 | 技术 |
|------|------|
| 服务端 | C++ (Windows) |
| 通信协议 | TCP |
| 数据库 | MySQL |
| 网络库 | Winsock2 |

## 快速开始

### 构建

**VSCode:** 按 `Ctrl+Shift+B`

**CMake:**
```bash
mkdir build && cd build
cmake ..
cmake --build .
```

### 运行

1. 确保 MySQL 服务运行中
2. 配置数据库连接（在 `Server/Kernel/kernel.cpp` 中修改）
3. 运行 `Server\Server.exe`
4. 服务端监听端口 **8899**

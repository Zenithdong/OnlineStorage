# 网盘项目 (OnlineStorage)

## 项目简介

这是一个跨平台的网盘系统项目，包含**客户端**和**服务端**两部分。

### 整体架构

```
┌─────────────────────────────────────────────────────────────┐
│                          客户端                              │
│              Android    Windows    Web                       │
└────────────────────────────┬────────────────────────────────┘
                             │  TCP / HTTP
┌────────────────────────────▼────────────────────────────────┐
│                          服务端                              │
│                       Server (本项目)                        │
└────────────────────────────┬────────────────────────────────┘
                             │
          ┌──────────────────┼──────────────────┐
          ▼                  ▼                  ▼
    ┌──────────┐      ┌──────────┐       ┌──────────┐
    │  MySQL   │      │  Redis   │       │ MongoDB  │
    │ (云数据库)│      │ (云数据库)│       │ (云数据库)│
    └──────────┘      └──────────┘       └──────────┘
                             │
          ┌──────────────────┼──────────────────┐
          ▼                  ▼                  ▼
    ┌──────────┐      ┌──────────┐       ┌──────────┐
    │  本地存储  │      │   OSS    │       │  Minio   │
    │ (文件存储) │      │ (阿里云OSS)│       │  集群    │
    └──────────┘      └──────────┘       └──────────┘
```

### 技术栈

| 组件 | 技术 |
|------|------|
| 客户端 | Android / Windows / Web |
| 服务端 | C++ (Windows) |
| 通信协议 | TCP / HTTP |
| 数据库 | MySQL / Redis / MongoDB |
| 文件存储 | 本地存储 / OSS / 阿里云OSS / Minio集群 |

## 项目结构

```
OnlineStorage/
├── Server/                    # 服务端项目 (本仓库)
│   └── (服务端代码和文档)
├── Client-Android/            # Android 客户端 (待开发)
├── Client-Windows/           # Windows 客户端 (待开发)
├── Client-Web/                # Web 客户端 (待开发)
└── README.md
```

## 服务端详情

服务端代码位于 `Server/` 目录，详细文档请查看 [Server/README.md](Server/README.md)。

## 快速开始

### 服务端构建

1. 用 VSCode 打开本项目
2. 按 `Ctrl+Shift+B` 构建
3. 运行 `Server\Server.exe`
